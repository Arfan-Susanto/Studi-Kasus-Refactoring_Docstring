# [INF2143] Praktikum Proyek SOLID OOP Python
## Pertemuan 12: Dokumentasi dan Version Control

### Deskripsi Proyek
Studi kasus pada proyek ini mendemonstrasikan implementasi prinsip SOLID (SRP, OCP, DIP) pada sistem transaksi dengan menggunakan Abstraction (Interface) di kelas induk dan Depedency Injection dengan contohnya Cafe.

### Struktur File
- `Studi_kasus_refactor.py`: Kode inti yang sudah direfaktor dan ditambahkan logging.
- `README.md`: Dokumen ini.

### Cara Menjalankan 
1. Pastikan Python 3.x (sekian) terinstal.
2. Jalankan file utama dari terminal:
    ```bash
    Studi_kasus_refactor.py
    ```
